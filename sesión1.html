Sesión 1

Comenzamos con el curso de Laboratorio de innovación 2 presentándonos y viendo el contenido del sílabo. Después de la pequeña introducción empezamos a ver los primeros conceptos:

HTML (Hyper Text Markup Language / lenguaje de marcado de hipertexto) HTML no es un lenguaje de programación. HTML es un lenguajes de marcado o de marcas, compuesto por etiquetas que se emplean para crear la estructura completa de una página web, desde la cabecera (head) hasta el pie de página (footer).

CSS (Cascading Style Sheets/ Hojas de estilo en cascada) CSS tampoco es un lenguaje de programación sino que es una tecnología que se emplea para darle estilo a una página web construida en HTML. Gracias a CSS se puede hacer una separación entre el contenido y estructura de la web y su parte visual o estética.

Hipertexto: es una referencia de una pagina a otra. Un sitio web está conformado por una enorme cantidad de páginas web.

Atributo de las etiquetas: Permiten dar propiedades a etiquetas específicas de html.

La etiqueta meta no se cierra. Se utiliza para usar sus atributos. Aunque la etiqueta meta no se vea, ayuda a que el sitio web se encuentre con más facilidad porque estás dando más información.
